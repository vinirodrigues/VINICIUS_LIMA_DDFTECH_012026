# Dataset – Olist Brazilian E-Commerce

## Fonte
- Nome: Olist Brazilian E-Commerce Dataset
- Origem: Kaggle
- Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Descrição Geral
Dataset público contendo dados de um e-commerce brasileiro, incluindo pedidos, clientes, produtos, vendedores, pagamentos e avaliações.

## Volume de Dados
- Número total de arquivos CSV: (preencher)
- Número aproximado de registros: +100.000

## Principais Arquivos
- olist_orders_dataset.csv – informações dos pedidos
- olist_customers_dataset.csv – dados dos clientes
- olist_order_items_dataset.csv – itens por pedido
- olist_products_dataset.csv – catálogo de produtos
- olist_order_payments_dataset.csv – pagamentos
- olist_order_reviews_dataset.csv – avaliações

## Observações Iniciais
- Dataset relacional, com chaves como `order_id`, `customer_id` e `product_id`
- Adequado para análises de vendas, logística e comportamento do cliente
